# Pasta de Assets

Esta pasta deve conter recursos adicionais do site como:

## Sugeridos:
- `favicon.png` - Ícone do site (32x32px ou 64x64px)
- `logo.png` - Logo da lancheria (opcional)
- Fontes personalizadas (se necessário)
- Ícones adicionais

## Nota:
O site já possui emojis integrados e funciona sem assets adicionais.
Adicione arquivos aqui conforme necessário para personalizar o visual.
